# webpack5-react-ts-mobx5-pc-demo
> webpack5-react-ts-mobx5-pc-demo

### 技术栈
webpack5
react18
react-router6
typescript5
mobx5



### 项目运行
- 全局安装包安装
- npm i -g cross-env
- 安装相应依赖包
- npm i
- 开发环境运行
- npm run dev
- 生成项目打包文件
- npm run build


### git-check
npx husky add .husky/pre-commit 'npm run pre-check'
